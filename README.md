# Semiconductor Wafer Defect Classification

AI-based semiconductor wafer defect classification using the WM-811K dataset, with baseline ML and CNN experiments.

## Overview
This project explores machine learning and deep learning approaches for semiconductor wafer defect classification using wafer map data.

## Methods
- Baseline Random Forest classifier
- Multiclass CNN classifier
- Improved lightweight CNN experiments

## Dataset
- WM-811K wafer map dataset

## Outputs
The project generates:
- training history plots
- confusion matrix plots
- sample prediction visualizations

## Note
Dataset files are not included in this repository due to size.
